《剑指offer》- 用两个栈实现队列

2016-11-02 • 分类: 算法 • 标签: 算法

题目描述：用两个栈实现一个队列，实现对了的两个函数 add 和 del, query，分别完成在队列尾插入结点和在队列头部删除结点的功能。

代码：

public class aa<T> {
    private Stack<T> stack1 = new Stack<T>();
    private Stack<T> stack2 = new Stack<T>();

    public  void  add(T t){
        stack1.push(t);
    }
    public  T  del  (){
        if(stack2.isEmpty()){
            while(!stack1.isEmpty()){
                stack2.push(stack1.pop());
            }
        }
        if(stack2.isEmpty()){
            System.out.println("当前栈为空");
        }
        return stack2.pop();
    }
    public void query(){
       while (!stack2.isEmpty()){
           System.out.println(stack2.pop());
       }
    }
    public static void main(String args[]) throws Exception  {
        aa<String> aa=new aa<>();
        aa.add("1");
        aa.add("2");
        aa.add("3");
        aa.add("4");
        aa.add("5");
        aa.del();
        aa.query();
    }
}

主要思想：先用stack1来接收传来的值,然后将stack1的值导入stack2中。将stack2出栈，遍历stack2.
