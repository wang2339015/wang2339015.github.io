j2EE常用框架

2016-05-26 • 分类: j2EE • 标签: j2EE

1.JDBC:Java数据库连接对象，基础中的基础，Hibernate也只是对它的封装而已
2.JNDI: Java 命名与目录接口，J2EE重要规范之一
3.EJBS: 上面已经提到过了
4.RMI: 提供远程调用方法的支持，主要用于程序分布式开发
5.JMS: Java消息服务，中间件技术
6.JAVA IDL: 提供J2EE平台与CORBA交互能力和互联能力的技术
7.JTS: 组件事务监视器
8.JTA: 事务划分的一个技术
9.JAVAMAIL: Java中的邮件技术
10.JAF: Java启动框架
11.Log4j，一款日志处理的框架应用
12.Junit：单元测试的好帮手
13.freemarker、velocity：两款不错的模板引擎
14.与Flash的交互Flex：目前很多的SNS应用就是使用到了这个技术
15.Web services:想知道我们论坛天气预报的效果是怎么做的吗？请关注这个技术
16.SOA:面向服务架构，未来技术发展的趋势之一
